High-Budget Films

Overview

This project compares the most expensive movies of all time based on six metrics: production budget, domestic gross, worldwide gross, total profit, return on investment, and release date. From a list of 1,000 movies, the top 20 movies for each metric are identified and compared side-by-side. For each film in the top 20, we also compare genre and film ratings as these are not provided in the original dataset.

The goal of this study is to identify the most expensive movie types to produce, the genres that perform best, and any similarities between the top-grossing movies. This information can be used to spot trends in the film industry. For example, we can determine if there is a positive correlation between production budget and worldwide gross, track how production budgets have been increasing over time, and recommend filmmakers to focus on genres with a high return on investment.

Data Sources

The data used for this study was collected from The Numbers, a film-industry data website that tracks box office revenue. The website is able to provide data on 6,400 movies with the highest production budgets. However, only the top 1,000 movies with the highest production budgets were used in this study. The data was retrieved using Microsoft Excel's built-in web scraping functionality to import every data column provided.

Data Cleaning & Modifications

The data retrieved from The Numbers was clean and free of errors. However, it is important to note that not all metric data for each movie is made public. In these cases, The Numbers defaults to $0 for any film without publicly available sales data. Additionally, the list of films includes those with the highest production budgets, even if they have not yet been released. For instance, Mission: Impossible Dead Reckoning is the 11th most expensive film, but it is not scheduled to be released until June 19, 2023. For this study, we are comparing movies based on all of the provided metrics. Therefore, any entries with missing data or unreleased films have been removed from the dataset.

The data retrieved from The Numbers website did not include film genres or ratings. This is unfortunate, as this information could be used to compare the top-performing films. As a result, I retrieved the genre and rating for the top 20 films in each metric (production budget, worldwide gross, domestic gross, return on investment, and total profit). While it would have been ideal to have this data for all films, it would have been too time-consuming and tedious to collect.
